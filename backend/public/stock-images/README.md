# Stock Images Directory

Place product images here to make them available in the Stock Gallery tab when editing products.

Supported formats: JPG, JPEG, PNG, WebP, GIF

The images will be accessible via the "📂 Stock" tab in the product image uploader.

## Recommended naming convention:
- pizza-margherita.jpg
- burger-classic.jpg
- salad-greek.jpg
- coffee-espresso.jpg
etc.

## Recommended dimensions:
- Square (1:1 ratio): 800x800px or higher
- This ensures consistent display in product grids
